Geral

O banco de dados completo é exportado, um arquivo para cada tabela. Isso significa que os dados de um abastecimento é separado em vários arquivos. Você pode encontrar o carro procurando o valor da coluna 'car' na coluna Id da tabela 'cars'.

Colunas

car

_id: [int]
car__name: [string]
color: [int] Procure na documentação de cores do Android para uma explicação mais detalhada de como o valor é calculado.
initial_mileage: [int] Quilometragem inicial do carro, quando iniciar o uso do aplicativo.
suspended_since: [date]
buying_price: [float] Preço de compra do veículo.
num_tires: [int] Número de pneus do veículo.

fuel_type

_id: [int]
fuel_type__name: [string]
category: [string]

other_cost

_id: [int]
car_id: [int] o Id do carro.
mileage: [int] Distância total (odômetro).
price: [float]
recurrence_interval: [int] 0 = uma vez, 1 = diário, 2 = mensal, 3 = trimestral, 4 = anual
recurrence_multiplier: [int] Indica o ciclo de recorrência em conjunto com o recurrence_interval. Exemplo: intervalo = 2 e multiplicador = 5 significa a cada 5 meses.
title: [string]
date: [date]
note: [string]
end_date: [date] Fim da recorrência nesta data.

refueling

_id: [int]
partial: [bool] (1 ou 0)
date: [date]
note: [string]
volume: [float]
fuel_type_id: [int] o Id do tipo de combustível.
station_id: [int] o Id da estação de serviço.
price: [float]
mileage: [int] Distância total (odômetro).
car_id: [int] o Id do carro.

reminder

_id: [int]
title: [string]
after_time_span_unit: [int] 0 = dias, 1 = meses, 2 = anos
after_time_span_count: [int] Indica um intervalo de tempo em conjunto com o after_time_span_unit. Exemplo: unit = 1 e count = 5 significa após 5 meses.
after_distance: [int]
start_date: [date]
start_mileage: [int]
car_id: [int] o Id do carro.
notification_dismissed: [bool] (1 ou 0)
snoozed_until: [date]

station

_id: [int]
station__name: [string] Nome da estação de serviço.

tire_list

_id: [int]
buy_date: [date]
trash_date: [date] Data de descarte.
price: [float]
quantity: [int]
manufacturer: [string]
model: [string]
note: [string]
car_id: [int] o Id do carro.

tire_usage

_id: [int]
distance_mount: [int] Quilometragem na montagem.
date_mount: [date]
distance_umount: [int] Quilometragem na desmontagem.
date_umount: [date]
tire_id: [int] o Id do pneu da tabela tire_list.
