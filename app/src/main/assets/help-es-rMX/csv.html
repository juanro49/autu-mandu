General

Toda la base de datos se exporta utilizando un solo archivo por cada tabla. Esto significa que la información de una recarga se divide entre varios archivos. Puedes averiguar a qué vehículo corresponde el archivo leyendo el valor de la fila id en de la tabla cars de la columna cars

Columnas

car

_id: [int]
car__name: [string]
color: [int] Consulta la documentación de la clase color de Android para una explicación más detallada de cómo se calcula este valor.
initial_mileage: [int] El kilometraje inicial del vehículo al momento de registrarlo en la aplicación.
suspended_since: [date]
buying_price: [float] Precio de compra del vehículo.
num_tires: [int] Número de neumáticos del vehículo.

fuel_type

_id: [int]
fuel_type__name: [string]
category: [string]

other_cost

_id: [int]
car_id: [int] el ID del vehículo.
mileage: [int] Distancia total (odómetro).
price: [float]
recurrence_interval: [int] 0 = una vez, 1 = diario, 2 = mensual, 3 = trimestral, 4 = anual
recurrence_multiplier: [int] Especifica la recurrencia junto con recurrence_interval. Ejemplo: recurrence_interval = 2 y recurrence_multiplier = 5 significa cada 5 meses.
title: [string]
date: [date]
note: [string]
end_date: [date] Fin de recurrencia.

refueling

_id: [int]
partial: [bool] (1 o 0)
date: [date]
note: [string]
volume: [float]
fuel_type_id: [int] el ID del tipo de combustible.
station_id: [int] el ID de la estación de servicio.
price: [float]
mileage: [int] Distancia total (odómetro).
car_id: [int] el ID del vehículo.

reminder

_id: [int]
title: [string]
after_time_span_unit: [int] 0 = días, 1 = meses, 2 = años
after_time_span_count: [int] Especifica un lapso de tiempo junto con after_time_span_unit. Ejemplo: after_time_span_unit = 1 y after_time_span_count = 5 significa después de 5 meses.
after_distance: [int]
start_date: [date]
start_mileage: [int]
car_id: [int] el ID del vehículo.
notification_dismissed: [bool] (1 o 0)
snoozed_until: [date]

station

_id: [int]
station__name: [string] Nombre de la estación de servicio.

tire_list

_id: [int]
buy_date: [date]
trash_date: [date] Fecha en que se desecharon.
price: [float]
quantity: [int]
manufacturer: [string]
model: [string]
note: [string]
car_id: [int] el ID del vehículo.

tire_usage

_id: [int]
distance_mount: [int] Kilometraje al montar.
date_mount: [date]
distance_umount: [int] Kilometraje al desmontar.
date_umount: [date]
tire_id: [int] el ID del neumático de la tabla tire_list.
