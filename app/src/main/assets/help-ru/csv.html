Общее

Вся база данных экспортируется по одному файлу на каждую таблицу. Это значит, что данные об одной заправке распределяются между несколькими файлами. Код авто можно найти в таблице cars в колонке Id.

Колонки

car

_id: [int]
car__name: [string]
color: [int] Подробнее в документации о цветах android, где описано, как вычисляются значения.
initial_mileage: [int] Начальный пробег авто при добавлении в приложение.
suspended_since: [date]
buying_price: [float] Цена покупки автомобиля.
num_tires: [int] Количество шин автомобиля.

fuel_type

_id: [int]
fuel_type__name: [string]
category: [string]

other_cost

_id: [int]
car_id: [int] код авто.
mileage: [int] Общий пробег (одометр).
price: [float]
recurrence_interval: [int] 0 = один раз, 1 = каждый день, 2 = каждый месяц, 3 = квартал, 4 = год
recurrence_multiplier: [int] множитель к recurrence_interval. Например: interval = 2 и multiplier = 5 означает раз в 5 месяцев.
title: [string]
date: [date]
note: [string]
end_date: [date] Дата окончания повторения.

refueling

_id: [int]
partial: [bool] (1 или 0)
date: [date]
note: [string]
volume: [float]
fuel_type_id: [int] код (Id) типа топлива.
station_id: [int] код (Id) заправочной станции.
price: [float]
mileage: [int] Общий пробег (одометр).
car_id: [int] код авто.

reminder

_id: [int]
title: [string]
after_time_span_unit: [int] 0 = дней, 1 = месяцев, 2 = лет
after_time_span_count: [int] Отрезок времени вместе с after_time_span_unit. Например: unit = 1 и count = 5 означает через 5 месяцев.
after_distance: [int]
start_date: [date]
start_mileage: [int]
car_id: [int] код авто.
notification_dismissed: [bool] (1 или 0)
snoozed_until: [date]

station

_id: [int]
station__name: [string] Название заправочной станции.

tire_list

_id: [int]
buy_date: [date]
trash_date: [date] Дата утилизации шин.
price: [float]
quantity: [int]
manufacturer: [string]
model: [string]
note: [string]
car_id: [int] код авто.

tire_usage

_id: [int]
distance_mount: [int] Пробег при установке.
date_mount: [date]
distance_umount: [int] Пробег при снятии.
date_umount: [date]
tire_id: [int] код шины из таблицы tire_list.
