Generale

L'intero database viene esportato un file per tabella. Questo significa che i dati di un rifornimento sono separati su file multipli. Puoi trovare l'auto cercando il valore della colonna auto nella colonna Id della tabella auto.

Colonne

car

_id: [int]
car__name: [string]
color: [int] Fai riferimento alla documentazione sui colori Android per una descrizione più dettagliata su come sono calcolati i valori.
initial_mileage: [int] Distanza iniziale percorsa dall'auto, quando si inizia ad usare l'applicazione.
suspended_since: [date]
buying_price: [float] Prezzo di acquisto del veicolo.
num_tires: [int] Numero di pneumatici del veicolo.

fuel_type

_id: [int]
fuel_type__name: [string]
category: [string]

other_cost

_id: [int]
car_id: [int] l'identificativo dell'auto.
mileage: [int] Distanza totale (contachilometri).
price: [float]
recurrence_interval: [int] 0 = una volta, 1 = giornaliero, 2 = mensile, 3 = trimestrale, 4 = annuale
recurrence_multiplier: [int] Specifica il periodi di ricorrenza insieme con recurrence_interval. Esempio: interval = 2 e multiplier = 5 significa ogni 5 mesi.
title: [string]
date: [date]
note: [string]
end_date: [date] Data di fine ricorrenza.

refueling

_id: [int]
partial: [bool] (1 o 0)
date: [date]
note: [string]
volume: [float]
fuel_type_id: [int] l'identificativo del tipo di carburante.
station_id: [int] l'identificativo della stazione di servizio.
price: [float]
mileage: [int] Distanza totale (contachilometri).
car_id: [int] l'identificativo dell'auto.

reminder

_id: [int]
title: [string]
after_time_span_unit: [int] 0 = giorni, 1 = mesi, 2 = anni
after_time_span_count: [int] Specifica un intervallo di tempo insieme a after_time_span_unit. Esempio: unit = 1 e count = 5 significa dopo 5 mesi.
after_distance: [int]
start_date: [date]
start_mileage: [int]
car_id: [int] l'identificativo dell'auto.
notification_dismissed: [bool] (1 o 0)
snoozed_until: [date]

station

_id: [int]
station__name: [string] Nome della stazione di servizio.

tire_list

_id: [int]
buy_date: [date]
trash_date: [date] Data di smaltimento.
price: [float]
quantity: [int]
manufacturer: [string]
model: [string]
note: [string]
car_id: [int] l'identificativo dell'auto.

tire_usage

_id: [int]
distance_mount: [int] Chilometraggio al montaggio.
date_mount: [date]
distance_umount: [int] Chilometraggio allo smontaggio.
date_umount: [date]
tire_id: [int] l'identificativo dello pneumatico dalla tabella tire_list.
